Colors

You can combine the colors with some classes, to change ANYTHING:

  • .bg-*-* to change background color.
  • .text-*-* to change text color.
  • .border-*-* to change border color.
  • .input-*-* to change the border color of text field

You can set the following properties only on :hover, and the input on :focus

  • .bg-*-*-h to change background color.
  • .text-*-*-h to change text color.
  • .border-*-*-h to change border color.
  • .input-*-*-f to change the border color of text field when FOCUSED
