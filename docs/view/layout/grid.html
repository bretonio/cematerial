Basic

The grid system is based in flexbox and is fully responsive. You need to set a container using the class .grid, and their children using the class .grid-col.

Gutters

You can set a gutter for the .grid using the classes:

  • .grid-gutter-xs - 4px gutters
  • .grid-gutter-sm - 8px gutters
  • .grid-gutter OR .grid-gutter-md - 16px gutters
  • .grid-gutter-lg - 32px gutters
  • .grid-gutter-xl - 64px gutters

Column Autofill

You can choose a column to fill the empty space of the grid using the class .col-fill in .grid-col element.

Columns Size

You can set a width for each column and for each breakpoint. Sized grid are based on 12 columns layout.
Better is the sum of numbers (*) for each line be equal to 12 to each breakpoint.
You can use the classes:

  • .xs-col-* - Breakpoint starts on smaller devices, use this for default sizes
  • .sm-col-* - Device width 600px or higher
  • .md-col-* - Device width 1024px or higher
  • .lg-col-* - Device width 1440px or higher
  • .xl-col-* - Device width 1920px or higher
