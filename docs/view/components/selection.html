Radio Button

To set styles for a radio selection, use the class .input in a input[type="radio"] element.

Feel free to use the classes text-*-* to set color when input are checked, defined on Color Palette.

Checkbox

Feel free to use the classes text-*-* to set color when input are checked, defined on Color Pallet.

Basic

To set styles for a checkbox selection, use the class .input in a input[type="checkbox"] element.

Switch

To set styles for a checkbox selection, use the class .input and .input-switch in a input[type="checkbox"] element.
