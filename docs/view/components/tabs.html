Basic

You create a tab-navigation using the class .tabs, with children .tabs-nav and .tabs-content.

  • Tabs are enable to use swipe (left and right). You can disable using the class .tabs-nowswipe on .tabs element.
  • .tabs-nav must have children links with [data-toggle="tabs"]. The current tab link, need the class .tab-active.
  • .tabs-list must have children containers with class .tab-content. The current tab, need the class .tab-visible.
  • The links on .tabs-nav and the contents in .tabs-list must have same order.

Colors

Feel free to use colors defined on Color Palette.

  • Add border-*-* to .tabs-nav element to change bottom mark color.
  • Add text-*-* to .tabs-nav element to change links text color.
