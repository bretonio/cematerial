Basic

Inputs (text fields) are defined by the class .input. You can use that in tags input, select and textarea

  • Using the .input insdie a label, you need to set a input label using class .label-text.

Grid

You can use the grid to group inputs and addons using the grid styles.

Colors

Feel free to use colors defined on Color Palette.

  • Use the class .input-*-* to change main color of an input.
  • Use the class .input-*-*-f to change the color of an input when focused.

Labels

Basic

Using the .input insdie a label, you need to add a class .label in label element, and set a input label using class .label-text.

  • To change label color when input are focused, you need to add the class .label-*-* using the colors defined on Color Palette.

Floating

To use floating label, you need use the input like basic instructions, and add a class .label-float to a label element.

  • If the input have a value, you need add the class .label-active to a label element.

Autogrow

You can make an input autogrow using the class .input-autogrow. It works for tags input and textarea
