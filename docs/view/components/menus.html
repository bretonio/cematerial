Colors

Feel free to use the classes bg-*-* for background color and text-*-* for text color, defined on Collor Pallet.

Also, you can use [data-class-hover] to combine with bg-*-* and text-*-* classes.

You can put a menu inside the dropdown, and set the position using the rules defined on Dropdown section.

Remember you can cascade the dropdown so submenus using the class .dropdown on li.
